
# API for DB gestion of IA 900 project

This API is used to manage the database of the IA 900 project.

# Installation

```bash
./setup.sh
```

# Usage

```bash
# Start the API
./app.sh start

# Stop the API
./app.sh stop

# Restart the API
./app.sh restart

# Run tests on the API
./app.sh test
```
